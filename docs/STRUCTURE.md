# OpenPGP Project Structure

## Core Directories

### `/core`
- `__init__.py`: Package initialization
- `advanced_crypto.py`: Cryptographic operations
- `blockchain_identity.py`: Blockchain identity management
- `file_sync.py`: Secure file synchronization
- `key_manager.py`: PGP key management
- `secure_file_sharing.py`: Secure file sharing
- `crypto_utils.py`: Crypto utilities
- `exceptions.py`: Custom exceptions
- `config.py`: Configuration
- `logger.py`: Logging utilities

### `/gui`
- `__init__.py`: Package initialization
- `main_window.py`: Main application window
- `key_manager_ui.py`: Key management UI
- `encryption_ui.py`: Encryption interface
- `identity_tab.py`: Identity management UI
- `file_share_ui.py`: File sharing UI
- `help.py`: Help system
- `menu.py`: Application menu
- `theme.py`: UI theming
- `version.py`: Version info
- `widgets/`: Custom UI components

### `/tests`
- Unit and integration tests
- Test configurations

### `/docs`
- Documentation files
- User guides
- API references

### `/assets`
- Icons and images
- Translation files
- Other static assets
